# Contributing

## Build

```sh
yarn build
```

## Test

```sh
yarn test
```

## Coverage

```sh
yarn coverage
```

An HTML coverage report should be generated at `./coverage/index.html`.

## Debugging

To debug in VS Code:

`Choose a file > Add a breakpoint > Press F5`

Read more: <https://vitest.dev/guide/debugging.html>

## Publish

```sh
yarn publish
```
